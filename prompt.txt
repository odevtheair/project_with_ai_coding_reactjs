กรอก PIN 6 หลัก

External API ตรวจสอบ PIN (123456)

ถ้าผิด → แสดง Error

คำแนะนำสำหรับระบบ Login ด้วย Express.js + JWT + MySQL

สถาปัตยกรรมระบบ
1.Backend (Express.js):
2.รับข้อมูล username/password
3.ตรวจสอบกับฐานข้อมูล MySQL
4.สร้าง JWT token เมื่อ login สำเร็จ
5.ส่ง token กลับไปให้ Frontend
6.MVC Pattern
7.แยก css ออกจาก js jsx 

Frontend (React/JSX):
1.ส่งข้อมูล login ไปยัง API ด้วย Axios
2.เก็บ JWT token (localStorage/sessionStorage)
3.ใช้ token ในการเรียก API อื่นๆ
4.MVC Pattern
5.แยก css ออกจาก js jsx 
6.LoginForm ตัดส่วน Navbar , Footer ออก
7.ปรับขนาด LoginForm ให้เล็กลง
8.PinCodeForm ตัดส่วน Logo/Brand ออก

คำแนะนำด้านความปลอดภัย :
1.เข้ารหัส Password: ใช้ bcrypt เสมอ
2.Validate ข้อมูล: ทั้ง Frontend และ Backend
3.Token Expiration: ตั้งเวลาหมดอายุที่เหมาะสม
4.Environment Variables: เก็บ secret key ใน .env
5.Rate Limiting: จำกัดจำนวนครั้งการ login
6.Refresh Token: สำหรับระบบที่ต้องการความปลอดภัยสูง
